Korture Hiring MCP

Paste a JD into Claude or ChatGPT, get the five behaviours that actually matter.

Korture Hiring is a remote Model Context Protocol server for hiring managers. It turns a job description into a behavioural measurement brief, the evidence behind each behaviour, demand levels, interview questions, and a population reality check on how rare this hire really is. Free to use with a Korture account.

MCP endpoint https://mcp.korture.com/api/mcp

What this server exposes

  • validate_jd — confirm the JD has enough signal to analyse.
  • create_brief — turn the JD into the full measurement brief.
  • enrich_brief — add depth based on follow-up answers.
  • get_brief — retrieve a brief by id.
  • get_dimensions — list the 15 behavioural dimensions Korture can measure.
  • get_external_population_check — how rare this hire really is in the labour market.
  • get_stats — aggregate stats about the Korture corpus.

After you have a brief

The MCP returns the brief and a link back to korture.com. Open it there to send the scan to candidates and review their results once they complete the assessment.
